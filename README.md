# HLL SPA Artillery Calculator

A web-based targeting calculator for Self-Propelled Artillery (SPA) in Hell Let Loose.

## Features

- Calculate accurate elevation settings for multiple SPA vehicles
- Support for British Bishop, Churchill AVRE, US Sherman M4A3, Soviet KV-2, and German Panzer III/Brummbär
- Interactive distance ruler with snap-to-grid functionality
- Auto-calculation mode or manual calculation
- Tank information with specifications and screenshots
- Military-themed UI with authentic styling

## How to Use

1. Select your tank/SPA vehicle
2. Enter target distance (200-600m range)
3. Enter height difference (negative if target is lower, positive if higher)
4. Enter your current elevation adjustment if needed
5. Get your calculated elevation setting

## Vehicles Supported

- **British (Bishop SP)** - 25-pounder self-propelled gun
- **British (Churchill AVRE)** - 290mm breaching vehicle
- **US (Sherman M4A3 105)** - 105mm close support tank
- **Soviet Union (KV-2)** - 152mm howitzer tank
- **DAK (Panzer III Ausf.N)** - 75mm L/24 infantry support
- **Germany (Sturmpanzer IV Brummbär)** - 150mm urban assault gun
